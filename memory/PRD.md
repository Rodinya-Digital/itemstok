# İtem Stok - Premium İçerik İndirme Platformu PRD

## Proje Özeti
Premium stok içerik indirme platformu (Envato Elements, Freepik, Shutterstock, Flaticon, Motion Array, Epidemic Sound). Laravel 8 + PHP + MySQL tabanlı.
